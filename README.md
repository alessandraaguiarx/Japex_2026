Modelagem Matricial do Quantitativo de Materiais

Sobre o Projeto

Este projeto tem como objetivo demonstrar a aplicação de conceitos de Álgebra Linear na Engenharia Civil, especificamente no cálculo de quantitativos de materiais em sistemas de alvenaria convencional e estrutural.

A proposta consiste em modelar matematicamente o consumo de materiais por meio de matrizes e transformações lineares, permitindo maior organização, escalabilidade e automação dos cálculos.
Fundamentação Matemática

O modelo é baseado na equação:

M = A · X

Onde:

- A = matriz de consumo de materiais
- X = vetor de entrada (área da obra)
- M = vetor de saída (quantitativo de materiais)

Essa modelagem representa uma transformação linear aplicada a um problema real da engenharia.
